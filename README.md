# UI 设计稿


## 在线预览

👉 [https://892959.github.io/juxing-ui-mockup/](https://892959.github.io/juxing-ui-mockup/)

## 功能特性

- **左侧工单详情**：工单信息、对话记录、底部回复区
- **右侧 AI Agent 面板**：Tab 切换（基本信息 / AI Agent / 历史相似工单 / 知识库）
- **历史相似工单卡片**：相似度评分、点赞/踩反馈、参考回复展开/收起
- **智能排序**：按点赞数降序排列

## 技术说明

- 纯 HTML + CSS + JS，无框架依赖
- 单文件部署，双击即可在浏览器中打开
- 通过 GitHub Pages 免费托管
- 字体源使用国内 CDN（fonts.loli.net）

## 本地使用

直接下载 `index.html`，双击打开即可，无需服务器环境。
